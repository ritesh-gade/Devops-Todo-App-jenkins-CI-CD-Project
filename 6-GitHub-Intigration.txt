2) How to change the point of repository by our URL
---> git remote set-url origin <Our URL>

3) How to check status
---> git status

4) Add our Current Repo in local
---> git add .

5) Again check status
---> git status

6) Then Commit Our repository
---> git commit -m "Added Server code"

7) Push our project to github
---> git push origin master
